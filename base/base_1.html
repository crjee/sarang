서체
구글에서 제공하는 Noto Sans KR을 사용하여 제작합니다.
  • @import url('https://fonts.googleapis.com/css2?family=Noto+Sans+KR:wght@100;400;500;700&display=swap');
  • 서체의 굵기는 100, 400, 500, 700을 사용하며, 400이 기본굵기 입니다.
  • 서체의 색상은 #666666이며, 헤드라인태그에 적용된 칼라는 #111111 입니다.

아이콘
FontAwesome에서 제공하는 아이콘의 Free버전을 사용합니다.
  • 링크 : <link rel="stylesheet" href="/common/css/all.css" />
  • font-family:"Font Awesome 6 Free";
  • Solid : font-weight:900 / Regular : font-weight:400
CSS 구성
base.css를 이용하여 통합관리 합니다.
  • base.css : 하위로 사용될 CSS파일을 import하는 방식으로 구성
  • reset.css : 태그의 기본속성을 정의
  • common.css : 모든 페이지에 공통으로 사용될 CSS 정의
  • cont.css : 사용자(User)모드에 사용되는 컨텐츠 CSS 정의
  • vari.css : 사용자(User)모드에 사용되는 컨텐츠 칼라 변형 CSS 정의
  • admin.css : 관리자(Admin)모드에 사용되는 컨텐츠 CSS 정의
  • etc.css : Spot성으로 사용되는 CSS 정의
JS 구성
Jquery 3.6.0 버전을 사용합니다.
  • 공개용 플러그인을 사용합니다.
  • common.js : 커스텀 JS를 구현
